    Type Name Description
    bool sessionPersistence

    bool useSafeMode

    If true, files are hashed, meaning that cached content cannot be switched out by external influences
    string cacheLocationOverride

    If set, the cache will be stored in the given place instead of the default path
    float daysValid

    Number of days that an entry in the cache should stay valid before requiring a re-download
